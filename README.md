# Student Marks Management System

This is a simple web project for managing student records and their marks.

---

## Features

* Register students
* Teacher/Admin login
* View students in dashboard
* Add and delete student records
* Enter marks (Math, Science, English)
* Calculate total, percentage, and grade
* Search student by roll number
* Show final result

---

## Technologies

* HTML, CSS, JavaScript
* PHP
* MySQL

---

## How to run

1. Install XAMPP
2. Start Apache and MySQL
3. Put the project folder inside `htdocs`
4. Open phpMyAdmin and create a database called `student_system`
5. Import the database file
6. Open in browser:
   http://localhost/student-marks-system/
