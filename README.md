# Student Marks Management System

This is a simple web project for managing student records and their marks.

---

## Features

* Register students
* Teacher/Admin login
* View students in dashboard
* Add and delete student records
* Enter marks (Math, Science, English)
* Calculate total, percentage, and grade
* Search student by roll number
* Show final result

---

## Technologies

* HTML, CSS, JavaScript
* PHP
* MySQL

---

## Notes

This project is built for learning purposes and kept simple.
